# 马油叶儿直播切片工作流

## 一、整体流程

获取直播素材 → ② 提炼卖点 → ③ 写切片脚本 → ④ 剪辑 → ⑤ 做封面 → ⑥ 发布+挂橱窗

---

## 二、卖点提炼五步法（每步不超过15字）

1. **面料** → "100支全棉，贴身穿像裹着云"
2. **版型** → "法式收腰剪裁，谁穿谁好看"
3. **搭配** → "搭西裤通勤、配牛仔休闲"
4. **身材适用** → "100-140斤都能驾驭"
5. **价格优势** → "专柜599，今天直播间只要XX"

---

## 三、视频结构模板（1分钟）

| 时间段 | 内容 | 说明 |
|--------|------|------|
| 0-3秒 | 钩子开头 | 上身亮相 / 一句话爆点 |
| 3-25秒 | 核心展示 | 面料细节 + 版型讲解 + 穿搭演示 |
| 25-45秒 | 深度说服 | 多场景搭配 / 对比 / 细节放大 |
| 45-55秒 | 尺码参考 | 适合什么身材、码数建议 |
| 55-60秒 | 转化引导 | "点左下角小黄车" |

## 三种钩子开头

- 穿搭展示型："这件衬衫配牛仔裤和裙子完全是两种感觉"
- 痛点解决型："微胖女生选上衣最怕显壮？这件收腰刚刚好"
- 细节放大型：直接给面料特写

## 四、剪辑参数

- 比例：9:16 竖屏
- 分辨率：1080P / 60帧
- BGM：音量约30%，不盖人声
- 字幕：黄色描边，关键卖点换色
- 转场：硬切为主，偶尔闪白
