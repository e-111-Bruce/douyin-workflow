## 一、AI 是怎么工作的

**一句话：AI 不是魔法，是超级规律发现器。**

人类把海量数据（整个互联网的文字、图片、代码）喂给它，它从这些数据里找规律。你问它问题，它根据学到的规律，一个字一个字地往外蹦——就像一个看过 1 万条带货视频的人，不用背公式也知道"讲面料要配特写"。

> 用哲学的话说：AI 就是在诸多有联系的事物上寻找共性。

---

## 二、API 是什么（全称：Application Programming Interface）

**一句话：API 就是程序之间的快递员。**

你（你的程序）→ 发请求 → API 快递员送到服务器 → 服务器处理完 → API 快递员送回结果 → 你收到数据

**实操验证：** 在终端（CMD）里敲一行命令，就能通过 API 查到厦门现在 21.4°C。

```
curl "https://api.open-meteo.com/v1/forecast?latitude=24.48&longitude=118.08&current=temperature_2m"
```

## 三、API 密钥是什么

**一句话：密钥 = 你的身份证 + 门禁卡 + 记账本。**

- 身份验证：证明你是你
- 权限控制：你能用哪些功能
- 计费：用了多少、该交多少钱
- ⚠️ 密钥就像银行卡密码，不能泄露

---

## 四、终端和 CMD 是什么

**一句话：黑窗口就是终端，CMD 是 Windows 自带的终端。**

- 终端 = "黑窗口"的总称，用文字命令控制电脑
- CMD = Windows 自带的那一种终端
- 不同系统有不同终端，但长得差不多，操作也差不多

**帮客户部署工具的公式：** 打开黑窗口 → 敲命令 → 回车

---

## 五、提示词（Prompt）

**一句话：提示词 = 剧本杀角色设定。**

你给 AI 的信息越详细，它越懂你。

| ❌ 随便写 | ✅ 认真写 |
|-----------|-----------|
| "帮我写个文案" | "法式衬衫，醋酸面料，25-35岁女性，60秒，痛点开头" |
| → 泛泛的，用不了 | → 直接能用 |

**万用模板：** "你是一个{角色}，要为{目标}做{任务}。要求：{具体要求}。格式：{输出格式}。"

## 六、部署是什么

**一句话：部署 = 搭好舞台让程序跑起来。**

```
拿代码 → 装依赖 → 配环境（填密钥） → 跑起来
剧本 → 拍电影 → 上映 → 观众能看
```

没部署的代码 = 没开门的餐厅。
